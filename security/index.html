Security and Privacy on mmap (client and server sides)

Prepared by: Marcy Regalado

Introduction

This assignment calls to test the security of mmap (client and server side) web application developed by Ming Chow for COMP 20: WebProgramming, Spring 2015. The web application enabled users to provide their location and then gather locations from other students in the class by utilizing the Google Maps API. From others locations, a user could see how far they were in comparison to others in the class; however, this web applicatoin still had vulnerabilities present. This document identifies vulnerabilities and recommendations for strenghtening the web application.

Methodology

Testing was completed by using with a "black box" method that is only available to the attacker. I ran the website on Firefox and used "Tamper Data", which is a modifer to HTTP/HTTPS headers and post paramerters. Using this tool I was able to find vulnerabiliteis throughout the code. Finally, I also reviewed the code to find mroe logic errors and opportunites for server-side expliotation.
